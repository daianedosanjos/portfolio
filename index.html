Portfólio

Burger Ladies

Uma interface intuitiva que permite a realização de pedidos por meio de um tablet, enviando-os diretamente para a cozinha. O sistema garante uma preparação organizada e eficiente dos pedidos, agilizando o atendimento e melhorando a experiência do cliente.

Olá devas

Uma rede social destinada a mulheres da tecnologia, proporcionando um ambiente para compartilhar experiências, conhecimentos e oportunidades de emprego. O objetivo é fomentar a colaboração e crescimento profissional, fortalecendo a presença feminina na indústria de tecnologia.

Jogo da memória

Uma aplicação web divertida e interativa, inspirada no clássico jogo de cartas. Desenvolvido com tecnologias modernas, desafia a memória do jogador ao encontrar pares corretos de cartas em um tabuleiro embaralhado. É uma experiência estimulante e cativante para todas as idades.

Data Lovers

Uma página web para visualizar conjuntos de dados que atendam às necessidades do usuário. Com uma interface intuitiva e interativa, os usuários podem explorar e filtrar os dados de forma personalizada, gerando insights valiosos para suas análises e tomadas de decisão.
